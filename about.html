What is Docker?

Docker is a platform which packages an application and all

its dependencies together in the form of containers


Click here to visit the Docker website

Docker has the ability to package an application and its dependencies into a virtual container that can run on any Linux, Windows, or MacOS computer. Docker can be used for fast, consistent delivery of your applications, responsive deployment and scaling, and running more workloads on the same hardware.

See below for a brief description of the components of Docker and Docker tools.

Components of Docker:

  • Software
    • The Docker daemon is called dockerd
    • Manages Docker containers and handles container objects
  • Objects
    • The main classes are images, containers, and services
  • Registries
    • Repository for Docker images

Docker Tools:

  • Docker Compose
    • Defines and runs multi-container Docker applications
  • Docker Swarm
    • Provides clustering functionality for Docker containers
    • Turns a group of Docker engines into a single Docker engine
