Basic Linux Commands

Linux commands are the essence of the entire Linux
operating system used to administer the entire system

cat - used to list the contents of a file on the standard output
cd - used to go to a directory
clear - clears the terminal screen
cp - copy files through the command line
echo - used to display a text or a string to the standard output/file
find - searches for files and directories
grep - find text in a file
history - shows previously used commands
locate - used to locate a file in the Linux system (just like search command in Windows)
ls - tells you what files are in the directory you are in
man & â€”help - to know more about a command and how to use it
mkdir - creates a folder or directory, rmdir - delete an empty directory
mv - move files through the command line
passwd - used to change the user account password
ping - used to test the reachability of a host on an IP network
pwd - tells you which directory you are in and gives absolute path
rm - delete files and directories
ssh - used to securely connect to a remote server/system
su - changes the shell to be used as a super user until you used the exit commands
sudo - to turn something as a super user
touch - create a file

To log into your Docker container:

Step 1. Install and enable the SSH service.

Step 2. Get the container's IP address by using docker inspect commands filtering out the results.

Step 3. Ping the IP address to make sure it's available.

Step 4. Use the SSH tool to connect to the image.

Step 5. Input password and connection will be established.
