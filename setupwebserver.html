Setting up a Web-Server with Docker

Setup a sample webserver via docker

Image Source

These are command to be followed to setup a webserver using docker

  • Step 1: Install Docker sudo apt install docker
  • Install and run Docker in your computer
  • Key-step: After the installation, you can run “​docker run hello-world​” to test your installation
  • Step 2: Pull an Ubuntu image from Dockerhub. docker pull ubuntu
  • Pull ubuntu and run it on your computer as a container.
  • Use “docker images” to check existing Docker images on your computer
  • Step 3: Install vim & apache2 sudo apt install apache2 vim
  • Install vim & apache2 in your container
  • un the Ubuntu image to start a Docker container, then log into this Ubuntu container: Command: ​docker run -it ubuntu:latest /bin/bash
  • Update system using“​apt-get update”​
  • Step 4: Share a directory between Docker and host. docker run -it -v /a_absolute_local_directory:/a_directory_in_Docker_container container_name /bin/bash
  • This will save any data after you shutdown the container
  • Step 5:Use vi to create and edit an index.html file in the shared directory. vi index.html
  • Use vi to create and edit a simple index.html file, then open it using your host browser
  • create an “index.html” file using “ vi index.html ”. Press “i” key to edit the new “index. Html” file. You can see “-- INSERT --” in the bottom of the command window, which means you can type html source code now. Try the shortcuts of vi to edit your file.
  • Step 6: use Docker to host a website. /etc/init.d/apache2 restart
  • Using a browser of you host computer to access “127.0.0.1”
