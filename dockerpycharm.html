Connect Docker with PyCharm

Connecting Docker with Pycharm allows you to write python code

How do you connect Docker to PyCharm?

To connect Docker to PyCharm, you'll first want to ensure that you have PyCharm Pro installed. You can do this by applying for the student license to get an activation code through the JetBrains website. Make sure you also have GIT installed, as well as Docker and ensure Docker is actively running. In PyCharm, go to VCS to checkout for version control and go to git and past your url to clone your repository and create a python script, if not done already.

Next, click on preferences/file settings and in the "build, execution, deployment" section, ensure that your docker build is running. To check this, click on the "+" button, fill out the information and as long as it's running, it will say "connection successful." Back on the PyCharm main screen, click "add configuration," located at the top right. Click the "+" button to add a new configuration and click on Dockerfile under Docker. Name your file, under Dockerfile choose "Dockerfile," name your image tag "barcode," and make sure run built image is checked off and press OK.

Lastly, press the play button (next to the configuration name button) and that will bring up the docker console inside the IDE. The image should come up under log.

The image to right will be the final outcome of the image looks like.
